ECMAScript 5 - Resumo

ECMAScript é uma linguagem de script padronizada pela Ecma International e é a especificação principal da linguagem JavaScript.

ECMAScript 5 foi finalizado há mais de cinco anos e é bem suportado em todos os navegadores modernos.

ECMAScript 5 é totalmente compatível com o Internet Explorer 10 e quase totalmente compatível com o IE9. As versões modernas do Chrome e do Firefox também oferecem suporte total ao ECMAScript 5.

Underscore.js é uma biblioteca JavaScript popular que contém “uma grande quantidade de auxiliares de programação funcional”. A maioria desses helpers trabalha em arrays e outros objetos enumeráveis ​​(coleções, como o Underscore as chama).

A especificação ECMAScript 5 inclui alguns desses auxiliares de programação funcional como métodos em objetos Array nativos do JavaScript.

indexOfe lastIndexOf, como _.indexOfe _.lastIndexOf, retorna o índice (ou último índice) no qual um valor pode ser encontrado na matriz

every, como _.every, retorna verdadeiro se todos os valores na matriz passarem no teste implementado pela função fornecida

some, como _.some, retorna verdadeiro se algum dos valores na matriz passar no teste implementado pela função fornecida

forEach, como _.each, executa uma função fornecida uma vez para cada item da matriz

map, como _.map, retorna uma nova matriz com os resultados da chamada de uma função fornecida em cada item da matriz

filter, como _.filter, retorna uma nova matriz com todos os itens da matriz que passam na função de teste fornecida incluídos.

reducee reduceRight, como _.reducee _.reduceRight, reduza uma lista de valores em um único valor usando uma função de acumulador (da esquerda para a direita ou da direita para a esquerda)

Aqui estão uma variedade de outros recursos que foram adicionados ao ECMAScript 5, além das funções auxiliares de matriz:

Object.keys, como _.keys, retorna uma matriz de propriedades enumeráveis ​​de um determinado objeto

Array.isArray, como _.isArray, retorna truese um determinado objeto é uma matriz

Date.now, como _.nowretorna o número de milissegundos decorridos desde 1º de janeiro de 1970 00:00:00 UTC

JSON.stringifyconverte objetos JavaScript em strings JSON e JSON.parseconverte strings JSON em objetos JavaScript.

Function.prototype.bind, como _.bind, retorna uma função que, quando chamada, tem seu thisargumento de palavra-chave definido para o valor fornecido

String.prototype.trim, retorna uma cópia da string com os espaços em branco removidos do início e do final da string

parseIntnão trata mais strings que começam com 0valores octais

Strict mode, Object.create, Object.seal, Object.freeze, getters/setters, e mais também foram adicionados no ECMAScript 5
